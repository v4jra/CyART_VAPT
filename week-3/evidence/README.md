# Evidence & Artifacts

This directory contains supporting evidence collected during the VAPT engagement.

## Evidence Types
- **screenshots/** – Proof of exploitation, shell access, and tool outputs
- **Raw-Data/** – Wireshark PCAP files capturing relevant traffic
- **hashes.txt** – SHA256 hashes to maintain evidence integrity

## Evidence Handling
All evidence was collected in a controlled manner and hashed to maintain chain-of-custody and data integrity.

Evidence files are referenced in the final PDF report using figure captions.
